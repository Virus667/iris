# IRIS — Telegram Chat Manager
**Мощный чат-менеджер для Telegram по подписке.** Модерация, статистика, антиспам, внутренняя экономика и десятки развлекательных модулей — всё в одном боте, который вы подключаете к своему чату за пару минут.

> 💎 Бот распространяется по подписке. Оформление и подключение — прямо в Telegram, без установки и настройки серверов.

---

## Почему IRIS

- **Ничего не нужно хостить.** Не надо арендовать сервер, ставить Python или возиться с БД — бот работает на нашей инфраструктуре.
- **Подключение за минуту.** Добавили в чат, оформили подписку в боте — готово.
- **Всё в одном.** Модерация, экономика, игры, статистика и антиспам без зоопарка из десяти разных ботов.
- **Постоянные обновления.** Новые модули и фиксы прилетают автоматически всем подписчикам.

## Возможности

Бот покрывает полный цикл управления сообществом — от базовой модерации до сложной внутричатовой экономики и игровых механик.

### Модерация
- **Ранговая система** — 6 уровней доступа (0 — участник, 5 — создатель). Вышестоящий модератор может отменять решения нижестоящих.
- **Наказания** — предупреждения (варны), мут, бан и кик с указанием причины и произвольным сроком (`{число} [минута|час|сутки|неделя|месяц|год]`).
- **Автоматизация** — настраиваемый лимит варнов, авто-бан по достижению лимита, сроки хранения наказаний.
- **Массовые операции** — амнистия, банлист, варнлист.
- **Завещание и передача прав** — восстановление доступа к чатам через резервный аккаунт.

### Триггеры и автонаказания
Система реакций бота на события в чате: автоматическое наказание за флуд/маты, изменение исхода команд, расширение логики стандартных действий.

### Гибкий доступ к командам (ДК)
- Настройка каждой команды по рангу, включение для всех или полное отключение.
- Личный доступ (ЛДК) — исключения для конкретных пользователей.
- Импорт настроек доступа из другого чата.
- Лог всех изменений доступа.

### Чистка чата
- Удаление сообщений (одиночное, массовое, «пург» по реплаю).
- Кик неактивных / активных / новичков / молчунов / удалённых аккаунтов.
- Кик по количеству сообщений за период.

### Настройка чата
- Смена названия, описания, закрепа, чат-ссылки (в т.ч. по заявкам).
- Правила и приветствие с переменными (`{имя}`, `{ж|м|мн}`).
- Автокик, минимальная регистрация, автозаявки.
- ТГ-администраторы, теги участников, управление разрешениями.
- Уведомления о входах/выходах, ограничения на ссылки, каналы и ботов.
- Антирейд и лимиты инвайтов.

### Сетка чатов
Объединение нескольких чатов в единую сеть для управления одной командой: глобальные баны, глобальные модераторы/админы, синхронизация доступа команд и прав.

### Профиль пользователя (анкета)
Описание, никнейм, звание, девиз, гражданство, пол, дата рождения, город, ачивки, подписки, регистрация во «вселенной».

### Статистика
- Активность чата и пользователей в виде графиков (сутки/неделя/месяц/всё время).
- Почасовая статистика, топы, олды, новички.
- Списки онлайн, граждан, VIP, неактивных, молчунов.
- Массовые упоминания и созывы.

### Экономика
- **Валюта** — передача, чеки (обычные и именные), история транзакций, порог передачи.
- **Инвестиционный актив** — биржа со стаканом заявок, покупка/продажа по рынку и лимиту, графики курса.
- **Коины** — валюта репутации чата, фарм (ферма), топы вложений.
- **VIP-статус** — расширенные функции, эмодзи в нике, личные РП-команды.
- **Бонусы** — банхаммер, плюсопад, минусит, анонимка.

### Игровые и развлекательные модули
- Дуэли (обычные и на ставку), Кубы, Русская рулетка.
- Кланы и Кружки — группировки участников с репутацией.
- Отношения и Браки — многоуровневые связи с опытом и HP.
- Репутация — плюсы/минусы, звёздный и золотой рейтинги.
- Награды — медали 8 степеней за заслуги.
- Шипперинг, погода, рандомайзеры (`!инфа`, `!выбери`, `!данет`, `!жребий`).
- РП-команды с возрастными категориями (0+, 13+, 18+).

### Утилиты
- **Закладки / Чатбук** — разметка важных дискуссий в чате.
- **Заметки** — структурированное хранение текста, ссылок и файлов.
- **Таймеры** — отложенные команды (от 1 минуты до 1 года).
- **Каталог** — размещение чата в публичном каталоге.
- **Голосования** — за команду и за бан (ГБ).
- **Репорты** — система жалоб с перенаправлением в админ-чат.
- **Розыгрыши** — конкурсы в чатах и каналах с разными режимами.
- **Инлайн-режим** — перевод валюты, биржа, погода и РП в любых чатах.
- **Бизнес-бот** — интеграция в личные сообщения.

## Обозначения параметров команд

| Обозначение | Значение |
|-------------|----------|
| `{число}` | настраиваемое значение (указывается без скобок) |
| `[минута\|час\|сутки]` | выбор одного из предложенных вариантов |
| `{период}` | сочетание `{число [минута\|час\|сутки\|неделя\|месяц\|год]}` |
| `{ссылка}` | ссылка на пользователя, @username или реплай |
| `{enter}` | перенос строки |

Команды поддерживают префиксы `!`, `.`, `/`, а также обращения по имени бота. Многие работают и без префикса.
